## Docker Image Building

> Backend (flask backend)
```bash
docker build --tag <project name> .
```

> Run Image in Container
```bash
docker run -it --gpus all -d -p 5000:5000 <image_name>
```

All done. Let's run it in "**localhost:5000**"
